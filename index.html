Política de Privacidade

Última atualização: 08 de maio de 2026

O aplicativo TapOS foi desenvolvido com o objetivo de oferecer uma experiência de teclado personalizada, respeitando totalmente a privacidade dos usuários.

1. Coleta de Informações

O TapOS não coleta, armazena ou compartilha qualquer tipo de informação pessoal do usuário.

Isso inclui, mas não se limita a:

- Textos digitados
- Senhas
- Dados pessoais
- Informações do dispositivo

2. Uso do Microfone

O aplicativo solicita permissão para acessar o microfone do dispositivo exclusivamente para a funcionalidade de digitação por voz.

O áudio capturado não é armazenado, gravado ou enviado para qualquer servidor, sendo utilizado apenas em tempo real para conversão em texto.

3. Uso de Internet

O TapOS não utiliza conexão com a internet para funcionamento e não transmite dados do usuário.

4. Compartilhamento de Dados

Nenhuma informação do usuário é compartilhada com terceiros, pois o aplicativo não coleta dados.

5. Segurança

Como o aplicativo não coleta dados, não há armazenamento de informações sensíveis, garantindo maior segurança ao usuário.

6. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada futuramente. Recomendamos que o usuário revise esta página periodicamente.
